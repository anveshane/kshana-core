## Task

Write the complete story based on the plot outline. The story should:

### Narrative Elements

1. **Rich Descriptions**: Provide vivid visual descriptions that translate well to images
2. **Character Voice**: Give each character distinct dialogue and mannerisms
3. **Emotional Beats**: Include clear emotional moments for each scene
4. **Visual Directions**: Embed visual cues that help with image generation

### Structure

- Organize into clear scenes (aim for 6-10 scenes)
- Each scene should have a clear beginning, middle, and end
- Include scene headings with location and time

### Writing Style

- Present tense, third person (cinematic style)
- Show, don't tell - focus on visible actions and expressions
- Include atmospheric details (lighting, weather, sounds)
- Keep dialogue concise and impactful

## Guidelines

- Target length: 1000-2000 words
- Each scene should be 100-200 words
- Balance description, action, and dialogue
- Ensure visual continuity between scenes
- End each scene with a hook or transition point

## Visual Writing Tips

When describing scenes, include:
- Camera angle suggestions (wide shot, close-up, etc.)
- Character positions and movements
- Lighting and color palette hints
- Key objects or props in frame
