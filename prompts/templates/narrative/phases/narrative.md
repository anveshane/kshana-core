# Story Writing Phase

This phase expands the approved plot outline into a complete narrative story.

## Phase Goal

Write a full narrative story with dialogue, descriptions, and emotional beats that will serve as the foundation for visual breakdown.

## Artifacts in This Phase

- **Full Story**: The complete narrative text

## Workflow

1. Review the approved plot outline
2. Expand each plot beat into fully written scenes
3. Develop character voices and dialogue
4. Add sensory details and emotional descriptions
5. Ensure the story flows naturally between beats
6. Present the complete story for user approval

## Writing Guidelines

### Scene Structure
- Each scene should have a clear purpose
- Include visual descriptions for image generation
- Balance dialogue and action
- Show character emotions through behavior

### Visual Storytelling
- Write with the camera in mind
- Describe what we SEE, not just what characters think
- Include environmental details
- Note lighting, weather, and atmosphere

### Dialogue
- Keep dialogue concise and purposeful
- Each line should reveal character or advance plot
- Consider that some scenes may have no dialogue

## User Interaction

Share drafts and ask for feedback on:
- Character voice and authenticity
- Pacing and scene length
- Emotional tone
- Any scenes that feel unclear visually

## Quality Criteria

Before completing this phase:
- [ ] All plot beats are expanded into scenes
- [ ] Characters have distinct voices
- [ ] Visual descriptions are clear enough for image generation
- [ ] Story has emotional resonance
- [ ] User has approved the full story
