## Image Requirements

1. **Consistency Focus**: This image will be used as a reference for all scenes featuring this character
2. **Clear Features**: Face, clothing, and distinctive features should be clearly visible
3. **Neutral Pose**: Use a neutral, front-facing or 3/4 pose for maximum utility as a reference
4. **Good Lighting**: Ensure the character is well-lit with details visible

## Prompt Construction

Create an image prompt that includes:
- Physical appearance details from the character description
- Distinctive features and accessories
- Clothing and style elements
- The style modifiers for visual consistency
- Neutral background (solid color or simple gradient)

## Negative Prompt Elements

{{STYLE_NEGATIVE_PROMPT}}
- Multiple people
- Busy background
- Motion blur
- Cropped face
- Text or watermarks

## Output

Generate the image and store it as the character's reference image.
