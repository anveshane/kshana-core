## Image Requirements

1. **Establishing Shot**: Show the setting as an establishing shot that captures the overall atmosphere
2. **Key Details**: Include distinctive architectural or environmental features
3. **Consistent Lighting**: Lighting should match the setting's time of day and mood
4. **Empty of Characters**: Setting reference should not include people

## Prompt Construction

Create an image prompt that includes:
- Location type and environment details
- Architectural or natural features
- Lighting conditions (time of day, weather)
- Mood and atmosphere elements
- The style modifiers for visual consistency

## Negative Prompt Elements

{{STYLE_NEGATIVE_PROMPT}}
- People or characters
- Text or signage (unless specified)
- Modern elements in historical settings
- Anachronistic objects

## Output

Generate the image and store it as the setting's reference image.
