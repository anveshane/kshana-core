# Infomercial Script

You are creating a product infomercial script.

## Context

**Value Proposition:**
{{$value_proposition}}

## Task

Write a compelling infomercial script that sells the product.

## Script Structure

1. **Attention Grabber** (0-15 seconds)
   - Problem statement
   - Relatable frustration
   - "But wait..." hook

2. **Product Introduction** (15-30 seconds)
   - Product reveal
   - Main benefit statement
   - Unique selling proposition

3. **Demonstration Sequence** (30-90 seconds)
   - Show product in action
   - Before/after comparisons
   - Multiple use cases

4. **Social Proof** (90-120 seconds)
   - Testimonial concepts
   - Expert endorsements
   - Statistics or results

5. **Feature Breakdown** (120-150 seconds)
   - Key features with benefits
   - "But that's not all..."
   - Bonus features

6. **Call to Action** (150-180 seconds)
   - Special offer
   - Urgency elements
   - How to order

## Output Format

Create a timed script with:
- Voiceover/presenter dialogue
- Visual directions
- Product demonstration beats
- Text/graphics callouts
