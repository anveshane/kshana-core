## Script Structure

### Opening Hook (0-10 seconds)
- Grab attention immediately
- Present the pain point
- Create curiosity

### Problem Amplification (10-30 seconds)
- Show the frustration
- "Tired of...?" moments
- Build relatability

### Solution Introduction (30-45 seconds)
- Introduce the product
- Position as the answer
- Key benefit statement

### Feature Demonstrations (45-120 seconds)
- Show key features working
- Proof of claims
- Multiple use cases

### Social Proof (if applicable)
- Testimonials
- Results shown
- Credibility builders

### Call to Action (final 15-30 seconds)
- Clear next step
- Urgency element
- Offer/pricing if relevant

## Writing Guidelines

### Tone
- Match brand voice
- Confident but not pushy
- Authentic and relatable

### Language
- Clear, simple words
- Active voice
- Benefit-focused language

### Pacing
- Quick to hook
- Time on demonstrations
- Strong close

## Workflow

1. Review value proposition
2. Plan script structure
3. Write each section
4. Add visual direction
5. Include timing notes
6. Present for approval

## Quality Criteria

Before completing this phase:
- [ ] Hook grabs attention
- [ ] Problem is relatable
- [ ] Solution is compelling
- [ ] Demos are planned
- [ ] CTA is clear
- [ ] User has approved
