# Value Proposition Development

You are creating a compelling value proposition for a product infomercial.

## Context

Use `list_project_files` to discover available content, then `read_file` to access what you need.

The original product information should be at `original_input.md`.

## Task

Develop a clear, persuasive value proposition that drives the entire infomercial.

## Value Proposition Components

1. **Target Audience**
   - Who is this product for?
   - What are their pain points?
   - What do they currently use/do?

2. **Problem Statement**
   - What problem does this solve?
   - Why do current solutions fail?
   - What's the cost of not solving it?

3. **Solution**
   - How does this product solve the problem?
   - What makes it unique?
   - Key differentiators from competitors

4. **Benefits**
   - Primary benefit (the main promise)
   - Secondary benefits (additional value)
   - Emotional benefits (how it makes them feel)

5. **Proof Points**
   - Social proof concepts
   - Demonstrations that prove claims
   - Before/after scenarios

6. **Call to Action**
   - What action should viewers take?
   - Why should they act now?
   - What's the offer?

## Output Format

Create a structured value proposition document that will guide all infomercial content.
