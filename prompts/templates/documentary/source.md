# Documentary Source Definition

You are defining a source (expert, organization, study, or interviewee) for a documentary.

## Context

Use `list_project_files` to discover available content, then `read_file` to access what you need.

The documentary outline should be at `plans/outline.md`.

## Task

Create a detailed source profile that can be used for:
- Visual representation (graphics, portraits)
- Attribution and credibility
- Planning interview questions or quotes

## Source Profile Requirements

1. **Identity**
   - Name and title
   - Organization/affiliation
   - Area of expertise

2. **Relevance**
   - Why this source matters for the documentary
   - What perspective or evidence they provide
   - Which segments they support

3. **Visual Representation**
   - How they should appear on screen
   - Suggested graphic style (portrait, name card, infographic)
   - Any associated data visualizations

4. **Key Points**
   - Main quotes or insights to highlight
   - Statistics or data they provide
   - Unique perspective they bring

## Output Format

Create a markdown profile with clear sections for each requirement.
