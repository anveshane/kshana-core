This phase creates the documentary's structure and argument flow.

Develop a comprehensive outline that organizes the documentary's content logically.

- **Research Outline**: The documentary's structural framework

## Outline Components

### 1. Opening Hook
- How will we grab the viewer's attention?
- What question or moment starts the journey?

### 2. Main Segments
- How many segments (typically 3-7)?
- What does each segment cover?
- How does each advance the thesis?

### 3. Evidence Structure
- What types of evidence will be used?
- Where will sources be integrated?
- How will claims be supported?

### 4. Narrative Arc
- How does the argument build?
- What's the emotional journey?
- Where are the key revelations?

### 5. Conclusion
- How does it all come together?
- What's the takeaway?
- What action or thought should viewers have?

## Workflow

1. Review the approved thesis
2. Identify the key points to cover
3. Organize into logical segments
4. Plan the evidence flow
5. Design the opening and closing
6. Present outline for approval

## Documentary Structure Types

### Chronological
- Events unfold in time order
- Good for historical topics

### Thematic
- Organized by themes or aspects
- Good for complex topics

### Problem-Solution
- Present the problem, then solutions
- Good for advocacy documentaries

### Investigation
- Follow the discovery process
- Good for mysteries or inquiries

## Quality Criteria

Before completing this phase:
- [ ] Outline logically supports the thesis
- [ ] Segments are balanced and coherent
- [ ] Evidence integration is planned
- [ ] Opening hooks viewer attention
- [ ] Conclusion provides satisfying closure
- [ ] User has approved the structure
