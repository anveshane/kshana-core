# Sources & Locations Phase

This phase identifies the sources, experts, and locations needed for the documentary.

## Phase Goal

Detail all sources for credibility and locations for visual storytelling.

## Current State

{{PHASE_STATE}}

## Input Context

Use `list_project_files` to see available content, then `read_file` to access what you need.

The thesis should be at `plans/thesis.md` and the outline at `plans/outline.md`. Read them to identify needed sources and locations.

## Artifacts in This Phase

- **Sources**: Experts, studies, organizations to reference
- **Locations**: Interview and b-roll locations

## Source Guidelines

### Types of Sources
- **Expert Interviews**: Academics, professionals, practitioners
- **Studies/Research**: Academic papers, reports, data
- **Organizations**: NGOs, companies, institutions
- **Primary Sources**: Documents, records, firsthand accounts

### Source Selection Criteria
- Credibility and expertise
- Relevance to thesis
- Visual representability
- Diversity of perspectives

### Source Documentation
For each source, define:
- Name and credentials
- Relevance to the documentary
- Key contributions or quotes
- Visual representation needs

## Location Guidelines

### Types of Locations
- **Interview Settings**: Where conversations take place
- **Establishing Shots**: Wide shots setting context
- **B-Roll Locations**: Supporting visual footage
- **Action Locations**: Where events happen/happened

### Location Selection Criteria
- Visual interest and atmosphere
- Relevance to content
- Accessibility for filming
- Emotional resonance

### Location Documentation
For each location, define:
- Physical description
- Atmospheric qualities
- Relevance to documentary
- Time of day/lighting needs

## Workflow

1. Review outline for source needs
2. Identify experts and evidence sources
3. Document each source thoroughly
4. Identify key locations needed
5. Document each location thoroughly
6. Present each for approval

## Quality Criteria

Before completing this phase:
- [ ] Sources support all major claims
- [ ] Diverse perspectives represented
- [ ] Locations enhance visual storytelling
- [ ] Each item individually approved
