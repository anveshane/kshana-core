# Segment Development Phase

This phase develops individual documentary segments with narration and visual direction.

## Phase Goal

Create detailed segment breakdowns that combine narrative, sources, and visual direction.

## Current State

{{PHASE_STATE}}

## Input Context

Use `list_project_files` to see available content, then `read_file` to access what you need.

The outline should be at `plans/outline.md`. Sources are in the `sources/` directory and locations in `locations/`. Read them to develop detailed segments.

## Artifacts in This Phase

- **Segments**: Individual documentary chapters

## Segment Structure

Each segment should include:

### 1. Segment Overview
- Title and duration estimate
- Main point or argument
- Role in overall documentary

### 2. Narration Script
- Voiceover text (if any)
- Key quotes from sources
- Transitions to/from segment

### 3. Visual Direction
- Shot list or visual suggestions
- Source graphics needed
- Location footage needed
- Timing notes

### 4. Evidence Integration
- Which sources are featured
- How evidence is presented
- Supporting visuals

## Workflow

For each segment:
1. Define the segment's purpose
2. Write the narration/script
3. Plan the visual accompaniment
4. Reference appropriate sources and locations
5. Present for approval

## Segment Guidelines

### Pacing
- Vary the rhythm within segments
- Balance information with engagement
- Allow breathing room for impact

### Visual-Audio Sync
- Visuals should enhance, not distract
- Key moments need strong visuals
- B-roll should be purposeful

### Source Integration
- Introduce sources appropriately
- Keep quotes concise and impactful
- Visual graphics for data

## Per-Item Approval

Each segment requires individual approval:
- Review content accuracy
- Check visual direction clarity
- Confirm source integration
- Approve before moving to visuals

## Quality Criteria

Before completing this phase:
- [ ] All segments advance the thesis
- [ ] Narration is clear and engaging
- [ ] Visual direction is specific
- [ ] Sources are properly integrated
- [ ] Each segment is individually approved
