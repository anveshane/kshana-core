# Documentary Outline Generation

You are creating a research outline for a documentary video.

## Context

**Thesis Statement:**
{{$thesis}}

## Task

Create a comprehensive outline that structures the documentary content logically and compellingly.

## Outline Requirements

1. **Opening Hook**
   - Capture audience attention immediately
   - Introduce the central question or thesis

2. **Background/Context**
   - Provide necessary historical or contextual information
   - Establish why this topic matters

3. **Main Segments** (3-5 segments)
   - Each segment should explore a distinct aspect of the thesis
   - Include potential sources, interviews, or evidence
   - Plan visual elements (B-roll, graphics, archival footage)

4. **Counterarguments/Complexity**
   - Address opposing viewpoints fairly
   - Show nuance and complexity

5. **Resolution/Call to Action**
   - Synthesize the evidence
   - Leave the audience with a clear takeaway

## Output Format

Create a markdown outline with:
- Clear section headings
- Bullet points for key content in each section
- Notes on visual elements needed
- Estimated duration for each segment
- Sources or interviews to include
