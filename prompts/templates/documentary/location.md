# Documentary Location Definition

You are defining a location or B-roll concept for a documentary.

## Context

Use `list_project_files` to discover available content, then `read_file` to access what you need.

The documentary outline should be at `plans/outline.md`.

## Task

Create a detailed location description for visual generation.

## Location Requirements

1. **Type**
   - Interview location
   - Establishing shot
   - B-roll footage
   - Historical/archival recreation

2. **Visual Description**
   - Physical environment
   - Lighting and atmosphere
   - Key elements to include
   - Mood and tone

3. **Purpose**
   - What story element this supports
   - Emotional impact desired
   - Connection to thesis

4. **Technical Notes**
   - Time of day
   - Weather/conditions
   - Camera angle suggestions
   - Duration needed

## Output Format

Create a markdown description with sections for visual generation prompts.
