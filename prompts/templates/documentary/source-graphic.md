## Graphic Types

Depending on the source type, generate:

### For Experts/Interviewees
- Professional portrait or headshot style
- Neutral background with professional feel
- Name and title can be added as overlay later

### For Organizations/Studies
- Logo-style representation or symbolic imagery
- Clean, professional graphic
- Infographic elements if representing data

### For Data/Statistics
- Clean data visualization
- Charts, graphs, or infographic elements
- Clear and readable presentation

## Image Requirements

1. **Professional Quality**: Broadcast/documentary standard
2. **Clear Communication**: The graphic should quickly convey the source's nature
3. **Consistent Style**: Match the documentary's visual language
4. **Neutral Presentation**: Avoid bias in visual representation

## Prompt Construction

Include:
- Type of graphic needed (portrait, chart, infographic)
- Key visual elements
- Color scheme appropriate to documentary style
- Professional and credible appearance
- Style modifiers for consistency
