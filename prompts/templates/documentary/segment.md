## Task

Create a fully scripted segment that can be produced as a video clip.

## Segment Requirements

1. **Opening**
   - Hook for this segment
   - Transition from previous content

2. **Content**
   - Narration script (voiceover text)
   - Interview clips or quotes to include
   - Facts and evidence to present
   - Visual directions for each beat

3. **Sources Used**
   - Which sources appear in this segment
   - How they are represented
   - Quotes or data attributed to them

4. **Visual Plan**
   - Shot-by-shot visual description
   - B-roll sequences
   - Graphics or animations needed
   - Transitions

5. **Closing**
   - Segment conclusion
   - Setup for next segment

## Output Format

Create a markdown script with:
- Narration in plain text
- Visual directions in [brackets]
- Source attributions clearly marked
- Estimated timing
