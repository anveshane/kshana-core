# YouTube Short Hook Development

Create an attention-grabbing hook for a short-form video.

## Input

{{ORIGINAL_INPUT}}

## Task

Develop a hook that captures attention in the first 1-2 seconds. Short-form content lives or dies by its hook.

### Hook Types

1. **Question Hook**
   - Poses an intriguing question
   - Creates curiosity gap
   - Example: "What happens when you mix these two chemicals?"

2. **Statement Hook**
   - Bold, surprising claim
   - Challenges expectations
   - Example: "Everything you know about productivity is wrong"

3. **POV Hook**
   - Places viewer in a scenario
   - Creates immersive experience
   - Example: "POV: You just discovered time travel"

4. **Visual Hook**
   - Striking or unusual visual
   - Action in progress
   - Example: Starting mid-action with satisfying result

5. **Conflict Hook**
   - Presents a problem
   - Creates tension
   - Example: "This one mistake ruins 90% of videos"

## Output Format

```markdown
# Hook

## Type
[Hook type used]

## Opening Line/Visual
[Exact text or visual description for first 1-2 seconds]

## Curiosity Gap
[What question/mystery does this create?]

## Promise
[What will viewers learn/see if they keep watching?]

## Payoff Setup
[How will the hook's promise be fulfilled?]

## Alternative Hooks
1. [Alternative hook option 1]
2. [Alternative hook option 2]
```

## Guidelines

- First 1-2 seconds are critical - lead with your best content
- Create a "curiosity gap" that demands resolution
- Avoid slow builds or context-setting - jump right in
- Consider visual + text combination for maximum impact
- Test multiple hook variations
