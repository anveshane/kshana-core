Write a tight, engaging script that delivers on the hook's promise.

## Script Structure

### Opening (0-3 seconds)
- Hook delivery
- Immediate engagement
- Visual and verbal alignment

### Body (3-40 seconds)
- Deliver the main content
- 1-3 key points maximum
- Fast pacing, no filler

### Close (last 5-10 seconds)
- Payoff or conclusion
- Call to action (optional)
- Satisfying ending or loop

## Script Components

For each beat, include:
- **Timing**: When it happens (e.g., "0:00-0:03")
- **Visual**: What's on screen
- **Audio**: Voiceover/text/music notes
- **Action**: What happens

## Writing Guidelines

### Brevity
- Every word must earn its place
- Cut ruthlessly
- Prefer showing over telling

### Rhythm
- Vary sentence length
- Create momentum
- Build to key moments

### Accessibility
- Works with sound off (text overlays)
- Clear visual storytelling
- Simple, direct language

## Total Duration

- Maximum: 60 seconds
- Optimal: 15-45 seconds
- Shorter is often better

## Workflow

1. Expand hook into full content
2. Structure the main beats
3. Write tight copy
4. Add visual direction
5. Time everything out
6. Present for approval

## Quality Criteria

Before completing this phase:
- [ ] Script delivers hook's promise
- [ ] Total duration under 60 seconds
- [ ] Every beat is necessary
- [ ] Visual direction is clear
- [ ] Works with sound off
- [ ] User has approved
