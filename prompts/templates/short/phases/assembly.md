## Assembly Process

### 1. Sequence
- Order clips per script timing
- Hook clip first
- Logical visual flow

### 2. Transitions
- Quick cuts (standard for shorts)
- Match cuts where possible
- Minimal fade usage

### 3. Timing
- Hit script beats precisely
- Maintain momentum
- No wasted frames

### 4. Duration Check
- Verify under 60 seconds
- Trim if needed
- Optimize for engagement

## Short-Form Assembly Best Practices

### The Hook
- First visual must grab
- No slow intros
- Immediate engagement

### Pacing
- Faster than you think
- Variety in shot length
- Build to key moments

### Ending
- Clear resolution OR
- Perfect loop point
- No awkward trailing

## Final Checks

### Technical
- 9:16 aspect ratio
- High quality output
- Clean audio (if any)

### Engagement
- Hook works instantly
- Middle delivers value
- End is satisfying

### Platform Ready
- Works on mute (text visible)
- Mobile optimized
- Thumbnail from strong frame
