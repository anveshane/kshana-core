# Visual Planning Phase

This phase plans the key visual moments for the Short.

## Phase Goal

Identify and describe the 3-5 key visuals that will make up the Short.

## Current State

{{PHASE_STATE}}

## Input Context

Use `list_project_files` to see available content, then `read_file` to access what you need.

The script should be available at `plans/script.md`. Read it to identify key visual moments.

## Artifacts in This Phase

- **Key Visuals**: Visual moment descriptions (max 5)

## Why Max 5 Visuals?

- Shorts are fast-paced
- Each visual needs impact time
- Simpler = more focused
- Quality over quantity

## Visual Types

### Hero Visual
- The main/most important shot
- Usually matches the hook
- Maximum impact

### Supporting Visuals
- Advance the content
- Illustrate points
- Build the narrative

### Transition Visuals
- Bridge between moments
- Create visual flow
- Can be simpler

## Per-Visual Documentation

For each key visual:

### Timing
- When in the script (e.g., "0:05-0:12")
- Duration

### Description
- What the viewer sees
- Composition and framing
- Key elements

### Text Overlay
- Any on-screen text
- Placement considerations
- Font style notes

### Motion Notes
- How will this animate?
- Energy level
- Transition style

## Workflow

1. Review script for visual beats
2. Identify must-have visuals
3. Consolidate to 5 or fewer
4. Detail each visual
5. Present for approval

## Vertical Composition

All visuals must work in 9:16:
- Center important elements
- Leave space for overlays
- Consider mobile viewing

## Quality Criteria

Before completing this phase:
- [ ] 5 or fewer visuals
- [ ] Each visual is necessary
- [ ] Timing matches script
- [ ] Vertical composition considered
- [ ] Each visual individually approved
