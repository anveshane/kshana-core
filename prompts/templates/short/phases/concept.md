# Hook Development Phase

This phase creates the attention-grabbing hook that starts the Short.

## Phase Goal

Develop a hook that stops the scroll and makes viewers want to watch.

## Current State

{{PHASE_STATE}}

## Artifacts in This Phase

- **Hook**: The attention-grabbing opener

## What Makes a Great Hook

### Immediate Value
- Promise something in the first 3 seconds
- Make it clear what they'll get
- Intrigue or satisfaction

### Types of Hooks

**Question Hook**
- "Did you know...?"
- "What if I told you...?"
- Creates curiosity

**Claim Hook**
- "This is the fastest way to..."
- "Nobody talks about this..."
- Bold statement

**Visual Hook**
- Something visually surprising
- Unusual or eye-catching
- Works without sound

**POV Hook**
- "POV: You discover..."
- Puts viewer in the situation
- Creates immersion

**Listicle Hook**
- "5 things you didn't know..."
- "3 reasons why..."
- Clear, numbered value

## Workflow

1. Analyze user's concept/idea
2. Identify the core appeal
3. Craft multiple hook options
4. Select the strongest hook
5. Present for approval

## Hook Checklist

- [ ] Works in first 3 seconds
- [ ] Clear value proposition
- [ ] Visually describable
- [ ] Platform appropriate
- [ ] True to content that follows

## Quality Criteria

Before completing this phase:
- [ ] Hook stops the scroll
- [ ] Promise is clear
- [ ] Can be visualized
- [ ] User has approved
