# YouTube Short Script

You are creating a script for a YouTube Short (under 60 seconds).

## Context

**Hook:**
{{$hook}}

## Task

Create a punchy, engaging script optimized for short-form vertical video.

## Script Requirements

1. **Timing**
   - Total duration: 15-60 seconds
   - Front-load the most engaging content
   - Every second counts

2. **Structure**
   - 0-3s: Hook (grab attention immediately)
   - 3-15s: Setup (establish context quickly)
   - 15-45s: Payoff (deliver the main value)
   - 45-60s: Call to action or punchline

3. **Visual Direction**
   - Each beat should have clear visual description
   - Consider text overlays
   - Plan for vertical format (9:16)

4. **Engagement Elements**
   - Pattern interrupts every 3-5 seconds
   - Visual or audio hooks
   - Satisfying conclusion or loop potential

## Output Format

Create a timed script:
```
[0:00-0:03] HOOK
Visual: [description]
Text: "..."
Audio: [voiceover/music/sfx]

[0:03-0:10] SETUP
...
```
