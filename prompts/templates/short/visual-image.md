## Image Requirements for Shorts

### Vertical Format
- Images should be optimized for 9:16 aspect ratio
- Key content centered for vertical viewing
- Composition works on mobile screens

### Attention-Grabbing
- High visual impact
- Clear focal point
- Works without context (scroll-stopping)

### Clarity
- Single clear subject/moment
- Not overly complex
- Readable at mobile size

## Prompt Construction

Include:
- Key visual elements from the description
- Vertical composition guidance
- Eye-catching elements
- Style modifiers for the aesthetic
- Text overlay considerations (leave space if needed)

## Composition for Vertical Video

### Center-Heavy
- Main subject in center third
- Safe area for overlays at top/bottom
- Works in feed preview

### High Contrast
- Clear subject separation
- Stands out in feed
- Works at small sizes

### Motion-Ready
- Consider how this will animate
- Leave room for movement
- Dynamic potential

## Negative Prompt Elements

{{STYLE_NEGATIVE_PROMPT}}

- Too much detail for mobile
- Cluttered or busy imagery
- Hard to read at small size
- Text or watermarks
