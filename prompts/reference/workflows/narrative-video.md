# Creating a Narrative Video

A narrative video brings a story to life through visuals and motion. This workflow transforms written narratives into engaging visual content with characters, settings, and scenes.

## What You Need to Create

A complete narrative video project requires these elements, built in sequence:

1. **Story**: Complete narrative with beginning, middle, end
2. **Characters**: Visual profiles for each person in the story
3. **Settings**: Where scenes take place (locations, environments)
4. **Scenes**: Key visual moments (typically 5-8 for a short video)
5. **Reference Images**: What characters and settings look like
6. **Scene Images**: The actual frames that will appear in the video
7. **Video Clips**: Animated scenes with motion
8. **Final Video**: Combined clips with transitions and audio

## Understanding the User's Input

Users provide input at different levels of completeness:

**Short Idea (1-3 sentences)**
- Needs development into a full story first
- Extract the core concept and genre
- Build out plot, characters, and scenes

**Complete Story (paragraphs, dialogue)**
- Ready to extract characters and settings
- Identify key visual moments for scenes
- Note any specific visual descriptions provided

**Partial Project (some elements exist)**
- Check project state to see what's done
- Continue from where things left off
- Maintain consistency with existing content

## The Creation Sequence

### Phase 1: Story Foundation

If the user provides only an idea, develop it into a complete story:
- Create the plot structure (beginning, conflict, resolution)
- Develop character arcs
- Build dialogue and emotional beats

### Phase 2: Visual Elements

Extract and create the visual building blocks:
- **Characters**: Name, appearance, personality, role in story
- **Settings**: Location, time of day, atmosphere, key visual details

Be specific. Vague descriptions make poor visuals. Instead of "a house," describe "a weathered Victorian house with peeling blue paint and overgrown garden."

### Phase 3: Scene Breakdown

Identify 5-8 key moments that capture the story visually:
- Each scene should advance the narrative
- Include the emotional tone
- Specify which characters and settings appear
- Describe the action and composition

### Phase 4: Visual Generation

Create images that bring the story to life:
1. First, generate reference images for characters and key settings
2. Use these references to maintain consistency
3. Generate scene images that match the established look

### Phase 5: Motion and Video

Transform still images into video:
- Add appropriate camera movement
- Create transitions between scenes
- Sync with narration or music if present

## Checkpoints

Get user approval at these points to avoid wasted effort:

**After Story Content**
- Plot, characters, and scenes should feel right before generating images
- Much easier to revise text than regenerate images

**Before Image Generation**
- Confirm the visual direction
- Images are expensive to generate

**After Key Images**
- Ensure characters look right before generating many scene variations
- Check setting consistency

**Before Video Generation**
- Video generation is the most expensive step
- Confirm image quality and sequence

## Handling Feedback

When users provide feedback on generated content:

- "Too dark" → Adjust the tone, lighting, or emotional weight
- "Doesn't match character" → Re-read character details, regenerate
- "Wrong setting" → Verify which setting applies to the scene
- "More dramatic" → Increase contrast, tension, or visual intensity
- "Simpler" → Reduce complexity, focus on core elements

## Tips for Quality

**Visual Consistency**
- Establish character appearance early with reference images
- Maintain consistent lighting and color palette within scenes
- Keep costume and setting details consistent

**Narrative Flow**
- Each scene should have a clear purpose
- Build emotional arc across the sequence
- End on a resonant moment

**Technical Considerations**
- Consider aspect ratio for intended platform
- Balance number of scenes with available context
- Complex scenes may need multiple images
