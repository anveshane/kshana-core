# Maintaining Visual Consistency

Visual consistency means characters, settings, and style remain recognizable across all images in a project. Without it, videos feel disjointed and unprofessional.

## Why Consistency Matters

**Recognition**
Viewers need to recognize the same character across scenes. If appearance changes dramatically, it's confusing.

**Immersion**
Inconsistency breaks the illusion. Each visual mismatch pulls viewers out of the story.

**Professionalism**
Consistent visuals signal quality. Inconsistency looks amateurish.

## Elements to Keep Consistent

### Characters

**Fixed Attributes**
- Face structure and features
- Body type and proportions
- Age appearance
- Distinctive marks (scars, birthmarks)

**Consistent Clothing**
- Core costume throughout (unless intentional change)
- Color palette of their wardrobe
- Style and accessories

**Hair and Makeup**
- Hairstyle and color
- Facial hair if applicable
- Makeup style

### Settings

**Architecture and Layout**
- Room shapes and sizes
- Furniture placement
- Architectural style

**Lighting Conditions**
- Time of day consistency
- Light source direction
- Shadow behavior

**Color Palette**
- Dominant colors of the space
- Wall, floor, ceiling colors
- Ambient color temperature

### Style

**Art Direction**
- Realistic vs. stylized
- Color grading approach
- Level of detail

**Rendering Style**
- Lighting approach
- Texture quality
- Edge treatment

**Mood and Atmosphere**
- Color temperature
- Contrast levels
- Overall feeling

## Strategies for Consistency

### Reference Images First

Before generating scenes, create reference images:
1. Character portraits (face, full body)
2. Key setting establishing shots
3. Style reference that captures the look

These become your "source of truth."

### Detailed Descriptions

Descriptions should include specifics:

**Weak**: "The detective enters the office."

**Strong**: "Detective Sarah Chen (Asian woman, late 30s, sharp features, black hair in a bun, gray wool coat over white blouse) enters her cluttered office (warm incandescent lighting, oak desk covered in papers, venetian blinds half-closed, brick wall visible through window)."

### Consistent Terminology

Use the same words for the same things:
- Always "Sarah Chen" not sometimes "Sarah" sometimes "the detective"
- Always "the oak desk" not sometimes "the wooden desk"
- Always "gray wool coat" not sometimes "dark coat"

### Style Anchors

Include style references in descriptions:
- Mention the overall visual style
- Reference established lighting
- Note color palette constraints

## Handling Changes

Sometimes things must change:

**Costume Changes**
- Establish why (new day, different context)
- Keep character features consistent
- May need new reference image

**Setting Changes**
- Clear transition in narrative
- Establish new setting clearly
- Maintain style consistency

**Time Passage**
- Show age appropriately
- Keep identifying features
- Core appearance still recognizable

## Common Consistency Problems

### Face Drift
Character faces change between images.
**Solution**: Use detailed facial descriptions, reference images, emphasize distinctive features.

### Costume Creep
Clothing changes unexpectedly.
**Solution**: Lock in costume description, repeat it exactly.

### Setting Shift
Same location looks different.
**Solution**: Establish setting thoroughly, reference specific elements.

### Style Wander
Visual style changes between images.
**Solution**: Strong style description, consistent art direction terms.

### Lighting Mismatch
Lighting direction changes impossibly.
**Solution**: Establish light sources, keep them consistent within scenes.

## Scene-Level Consistency

Within a single scene:
- Same time of day
- Same weather
- Same lighting direction
- Consistent shadows

Between scenes:
- Story logic for changes
- Character recognition maintained
- Overall style coherent

## Working with Limitations

Image generation has inherent variation. Strategies:

**Accept Minor Variation**
Small differences are often unnoticeable. Perfect consistency isn't required.

**Emphasize Distinctive Features**
The more unique the description, the more consistent results.

**Batch Related Images**
Generate images for one character/setting together.

**Iterate on Outliers**
If one image breaks consistency, regenerate it.

## Consistency Checklist

Before generating scene images, verify:

- [ ] Character reference images exist
- [ ] Descriptions use consistent terminology
- [ ] Clothing specified exactly
- [ ] Setting details locked in
- [ ] Style direction clear
- [ ] Lighting established

Before final video:

- [ ] Characters recognizable throughout
- [ ] Settings consistent within scenes
- [ ] No impossible continuity errors
- [ ] Style coherent across all images
